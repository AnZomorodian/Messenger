# OCHAT Messenger

A secure, real-time chat application with advanced privacy features.

## Key Features
- **Real-time Messaging**: Instant communication with polling.
- **Privacy First**: Local-only storage, no external database tracking.
- **Witness Mode**: Add a trusted third party to direct chats with mutual consent.
- **Rich Formatting**: Ctrl+B (Bold), Ctrl+I (Italic), Ctrl+U (Underline), Ctrl+H (Hyperlink).
- **Media Support**: 2MB images, 5MB files (auto-deleted for privacy).
- **Admin Control**: Secure management of users and global message history.
- **Customizable**: Dark/Light mode and user-defined color themes.

## Technical Stack
- **Frontend**: React, Vite, Tailwind CSS, shadcn/ui.
- **Backend**: Node.js, Express.
- **Storage**: MemStorage (In-memory, local only).
